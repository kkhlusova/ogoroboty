# Система управления роботом OmegaBot

Этот проект позволяет управлять роботом OmegaBot с помощью клавиш WASD с ноутбука оператора через Raspberry Pi 3 и Arduino Uno.

## Архитектура системы

- **Ноутбук оператора**: Отправляет команды через SSH на Raspberry Pi
- **Raspberry Pi 3**: Принимает команды от ноутбука и передает их по UART на Arduino
- **Arduino Uno**: Получает команды по UART и управляет двигателями робота

## Подключение двигателей к Arduino Uno

- **Левый двигатель движение**: Пин D6
- **Левый двигатель стоп**: Пин D7
- **Правый двигатель движение**: Пин D5
- **Правый двигатель стоп**: Пин D4

## Команды управления

- **W** - Движение вперед
- **S** - Движение назад
- **A** - Поворот влево
- **D** - Поворот вправо
- **X** - Остановка
- **Q** - Выход из программы

## Установка и настройка

### На Raspberry Pi:

1. Скомпилируйте программу:
   ```bash
   make
   ```

2. Запустите программу:
   ```bash
   ./robot_controller
   ```

3. (Опционально) Установите программу в систему:
   ```bash
   sudo make install
   ```

### На Arduino Uno:

1. Откройте файл `omegabot_control.cpp` в Arduino IDE
2. Загрузите программу на Arduino Uno
3. Убедитесь, что Arduino подключен к Raspberry Pi через USB

## Сборка проекта

Для компиляции программы управления для Raspberry Pi:
```bash
make all
```

Для очистки скомпилированных файлов:
```bash
make clean
```

## Примечания

- Убедитесь, что порт UART (/dev/ttyACM0) доступен и имеет соответствующие права доступа
- При первом запуске может потребоваться добавить пользователя в группу dialout: `sudo usermod -a -G dialout $USER`
- Скорость UART установлена на 9600 бод
